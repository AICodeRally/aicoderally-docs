## Module Architecture

Each module includes:

### 1. Module Definition
```typescript
// packages/modules/your-module/index.ts
export const module: RallyModule = {
  meta: {
    id: "your-module",
    name: "Your Module",
    description: "What it does",
    tier: "edge",
    category: "business",
  },
};
```

### 2. UI Page
```
apps/edge/app/modules/your-module/page.tsx
```

### 3. Data Models
```prisma
// prisma/schema.prisma
model YourModuleData {
  id        String   @id @default(uuid())
  // ... fields
}
```

### 4. API Routes (Optional)
```
apps/edge/app/api/your-module/route.ts
```

## Quick Start

### Browse Existing Modules
```bash
# View all modules
ls packages/modules/

# Check Edge modules
ls apps/edge/app/modules/
```

### Use a Module
1. Visit https://edge.aicoderally.com
2. Click the module card
3. Start using immediately (no setup required!)

### Customize a Module
1. Clone the repository
2. Navigate to `apps/edge/app/modules/your-module/`
3. Edit `page.tsx`
4. Deploy to your own Vercel project
