## Diagram 3: Compound Value Model

### Traditional Software Development

```
Year 1           Year 2           Year 3
┌──────────┐     ┌──────────┐     ┌──────────┐
│ Prototype│ ──▶ │   MVP    │ ──▶ │Enterprise│
└──────────┘     └──────────┘     └──────────┘
     ❌               ❌               ❌
 Throw away      Rebuild        Start over

Value: Linear, with resets
Cost:  High (rebuilds)
Time:  Slow (restarts)
```

### AICodeRally 3–6–∞ Model

```
Year 1           Year 2           Year 3
┌──────────┐     ┌──────────┐     ┌──────────┐
│  Studio  │ ──▶ │   Edge   │ ──▶ │  Summit  │
└──────────┘     └──────────┘     └──────────┘
     ✅               ✅               ✅
   Keep it        Add to it      Orchestrate

Value: Compounding (keeps everything)
Cost:  Lower (no rebuilds)
Time:  Faster (continuous evolution)
```

### Designer Notes
- Use red X's and green checkmarks prominently
- Show value curve increasing exponentially for AICodeRally
- Traditional model should show flat/declining value curve

## Designer Handoff Notes

### Color Palette

**Studio (App Layer):**
- Primary: Cyan `#06b6d4`
- Secondary: Teal `#14b8a6`
- Accent: Emerald `#10b981`
- Shape: `rounded-xl` (12px)

**Edge (Business Layer):**
- Primary: Orange `#f97316`
- Secondary: Rose `#f43f5e`
- Accent: Pink `#ec4899`
- Shape: `rounded-lg` (8px)

**Summit (Enterprise Layer):**
- Primary: Violet `#8b5cf6`
- Secondary: Fuchsia `#d946ef`
- Accent: Yellow `#eab308`
- Shape: `rounded-md` (6px)

**Modules (Capabilities):**
- Primary: Gray `#6b7280`
- Background: `#f3f4f6`
- Shape: `rounded` (4px)

### Typography

- **Headings:** Inter Bold
- **Body:** Inter Regular
- **Code/Technical:** JetBrains Mono

### Animation Guidelines

- **Tier transitions:** Fade up + scale (300ms ease)
- **Flow arrows:** Animated dash pattern
- **Checkpoints:** Bounce in when reached
- **Value curves:** Draw animation (1000ms ease-out)
